* 분석할수록 20~30대의 유저가 더 비중이 높다는것을 파악
* 예상했던 결과와 실제 결과의 차이점을 생각한 결과 어플이라는 접근성과 개인과 사업하는 유저들의 비교에서 사업하는 유저가 더 많이 보여서 40대이상의 연령대는 어플에 의존하기 보다 세무서에서 처리한다고 생각함
